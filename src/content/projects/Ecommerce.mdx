# Infraestructura E-commerce en Azure

Infraestructura diseñada para un sistema de comercio electrónico (e-commerce) en Azure, utilizando servicios escalables y gestionados.

---

## Descripción General
Arquitectura modular que incluye:
- **Frontend**: Aplicaciones web para usuarios (frontoffice) y administradores (backoffice).
- **Backend**: APIs y procesamiento asíncrono.
- **Almacenamiento**: Multimedia y colas de mensajería.
- **Base de Datos**: SQL Server para información transaccional.
- **Cache**: Redis para optimizar rendimiento.
- **Búsqueda**: Azure Cognitive Search para funcionalidades de búsqueda.
